## 目录结构

    data
      |--- 原始数据
    doc
      |--- 项目文档
    old_version
      |--- 之前项目的代码
    site
      |--- README.md  // 必读：描述了如何配置、启动前后端
      |--- frontend  // 前端项目
            |--- app  // js source
            		|--- app.js  // main entry
            |--- assets
                 |--- css
                 |--- images
            |--- node_modules  // 3rd party libraries
            |--- bundle // build directory
                 |---- bundel.js
      |--- backend  // 后端项目
    README.md
